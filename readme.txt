Installation:
  pip install streamlit pandas numpy reportlab pyyaml openpyxl

Lancement:
  streamlit run app_streamlit.py

Étapes:
  1) Onglet Créateurs: calcule palier 1/2, bonus débutant, statut actif.
  2) Onglet Agents: cumuls 2 mois si nécessaire, 2%-3% + bonus 1000/15000.
  3) Onglet Managers: idem agents, bonus 1000/5000.
  4) Exports CSV et PDF séparés pour chaque onglet.
